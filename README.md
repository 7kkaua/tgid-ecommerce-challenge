# 🛒 TGID Store | E-commerce Challenge

Este projeto foi desenvolvido como resolução do desafio técnico para a vaga de **Desenvolvedor Front-end Júnior** na TGID. A aplicação simula uma plataforma de e-commerce completa, consumindo uma API REST e focando em uma arquitetura escalável e UX premium.

## ✨ Funcionalidades Implementadas

- **Integração com API REST:** Consumo de dados assíncronos via JSON Server.
- **Amoled Dark Mode:** Alternância entre temas claro e escuro (Pure Black) com persistência via LocalStorage.
- **Visualização Individual:** Detalhes técnicos e especificações completas de cada produto através de Modais imersivos.
- **Carrinho Inteligente:** Gerenciamento de estado global com Context API, permitindo adicionar, remover e atualizar quantidades.
- **Barra de Busca:** Filtro dinâmico de produtos em tempo real com estados de "Empty Search" tratados.
- **Arquitetura Modular:** Organização de pastas por responsabilidades (Hooks, Utils, Services e Components).

## 🛠️ Stack Tecnológica

- **React.js** (Context API, Hooks, useEffect)
- **JSON Server** (Simulação de API REST)
- **Tailwind CSS** (Estilização e Dark Mode)
- **Lucide React** (Ícones)
- **Sonner** (Notificações Toast)

## 🚀 Como executar o projeto localmente

Como o projeto utiliza uma API simulada, você precisará de dois terminais abertos:

1. **Clone o repositório:**
   git clone [https://github.com/7kkaua/tgid-ecommerce-challenge.git](https://github.com/7kkaua/tgid-ecommerce-challenge.git)
   cd tgid-ecommerce-challenge

2. Acesse a pasta do projeto:
   cd tgid-ecommerce-challenge

3. Instale as dependências:
   npm install

4. **Inicie o servidor de desenvolvimento:**
   npm run dev
   
5. Acesse em seu navegador: http://localhost:5173
