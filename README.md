# 🛒 TGID Store | E-commerce Challenge

Este projeto foi desenvolvido como resolução do desafio técnico para a vaga de **Desenvolvedor Front-end Júnior** na TGID. Trata-se de uma plataforma de e-commerce de alta performance, focada em uma experiência de usuário (UX) premium e arquitetura de software escalável.

## ✨ Funcionalidades Implementadas

- **Amoled Dark Mode:** Alternância entre temas claro e escuro (Pure Black) com persistência de preferência no navegador via LocalStorage.
- **Vitrine Dinâmica:** Listagem de produtos com filtros em tempo real e estados de "Empty Search" personalizados.
- **Modal de Detalhes:** Visualização imersiva de especificações técnicas, dimensões e pesos de cada item.
- **Carrinho Inteligente (Drawer):** Gerenciamento completo de estado global com cálculos automáticos de subtotal e quantidades.
- **Checkout Simulado:** Fluxo de finalização de compra com tela de confirmação animada e feedback visual.
- **Arquitetura Clean Code:** Organização modular utilizando Hooks customizados, utilitários de formatação e componentização avançada para máxima manutenibilidade.

## 🛠️ Stack Tecnológica

- **React.js** (Context API para gerenciamento de estado global)
- **Vite** (Build Tool de alta performance)
- **Tailwind CSS** (Design System utilitário e Dark Mode via classes)
- **Lucide React** (Pacote de ícones vetoriais modernos)
- **Sonner** (Notificações toast premium para feedback do usuário)

## 🚀 Como executar o projeto localmente

1. **Clone este repositório:**
   git clone [https://github.com/7kkaua/tgid-ecommerce-challenge.git](https://github.com/7kkaua/tgid-ecommerce-challenge.git)

2. Acesse a pasta do projeto:
   cd tgid-ecommerce-challenge

3. Instale as dependências:
   npm install

4. **Inicie o servidor de desenvolvimento:**
   npm run dev
   
5. Acesse em seu navegador: http://localhost:5173
