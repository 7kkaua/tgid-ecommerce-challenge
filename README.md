# 🛒 TGID Store | E-commerce Challenge

Este projeto foi desenvolvido como resolução do desafio técnico para a vaga de **Desenvolvedor Front-end Júnior** na tgid. Trata-se de uma simulação de loja virtual com foco em usabilidade, performance e gerenciamento de estado.

## ✨ Funcionalidades

- **Vitrine de Produtos:** Listagem dinâmica de itens.
- **Filtro Inteligente:** Barra de busca em tempo real para encontrar produtos pelo nome.
- **Carrinho de Compras (Drawer):** Interface lateral moderna para gerenciamento de itens (adicionar, remover, alterar quantidade).
- **Persistência de Dados:** Uso de `localStorage` para manter o carrinho salvo mesmo após o recarregamento da página.
- **Feedback Visual:** Notificações (Toasts) amigáveis durante as interações do usuário.
- **Responsividade:** Layout totalmente adaptável para dispositivos móveis, tablets e desktops.

## 🛠️ Tecnologias e Ferramentas

- **React.js** (com Context API para gerenciamento de estado global)
- **Vite** (Build tool rápida e moderna)
- **Tailwind CSS** (Estilização utilitária e design system responsivo)
- **Lucide React** (Ícones modernos)
- **Sonner** (Biblioteca para Toast Notifications premium)

## 🚀 Como executar o projeto localmente

1. Clone este repositório:
   git clone [https://github.com/7kkaua/nome-do-seu-repositorio.git](https://github.com/7kkaua/nome-do-seu-repositorio.git)

2. Acesse a pasta do projeto:
   cd nome-do-seu-repositorio

3. Instale as dependências:
   npm install

4. Inicie o servidor de desenvolvimento:
   npm run dev

5. O aplicativo estará disponível em http://localhost:5173
